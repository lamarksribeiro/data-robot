# Data Robot

Robô de trading real do ecossistema GoldenLens para mercados Polymarket (BTC 5 minutos), operando na Polygon Mainnet via `@polymarket/clob-client-v2`.

Projeto em preparação. A implementação atual vive em `polymarket-robot` e será migrada para este repositório.

## Projetos relacionados

- `data-colector` — coleta e histórico de ticks
- `data-backtest` — lakehouse e backtests analíticos
- `clob-client-v2` — SDK CLOB da Polymarket
